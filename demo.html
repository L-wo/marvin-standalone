Marvin Standalone Example HTML Page

Give the above div(id="game") an attribute called "player-name" with your user's username using templating or JQuery or whatever you have available.

The player's name will be displayed at the end of the game. If you don't give the div that attribute, then that's fine too, it will omit the player name from the ending screen.

The game will also output an event to div(id="game") containing their name and score for you to use in a database if you wish.

View source to see an example JQuery handler for the score output event.
